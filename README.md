# Around-Me-Virtual
An iOS app that allows you to search locations on the map, add your current location and fetch images based on pins you drop on the map from the FlickrAPI

## Technologies Used:
CoreData <br>
CoreLocation <br>
SQLLite <br>
MapKit <br>
UICollectionView <br>

## Frameworks Used:
FlickrAPI <br>
FacebookGraphAPI <br>
Reachability <br>
